# astrbot-plugin-muyu

[![AstrBot](https://img.shields.io/badge/AstrBot-plugin-6c5ce7.svg)](https://github.com/AstrBotDevs/AstrBot)

赛博木鱼 —— 敲一下功德 +1，群聊解压神器。

## 为什么会有这个插件？

群聊需要一点无意义但快乐的东西。敲木鱼，攒功德，攀排行榜。没有复杂规则，不需要动脑，就是纯粹的快乐。适合摸鱼、解压、以及一切需要用"功德+1"来缓解的瞬间。

## 指令

| 指令 | 说明 |
|------|------|
| `/敲木鱼` | 敲一下，功德 +1 |
| `/功德排行榜` | 查看群内功德 Top 10 |
| `/今日功德` | 查看自己的功德数 |

## 里程碑成就

| 功德数 | 称号 |
|--------|------|
| 10 | 入门级善人 |
| 50 | 木鱼敲成铁鱼 |
| 100 | 赛博高僧 |
| 500 | 诺贝尔和平奖候补 |
| 1000 | 木鱼已碎 |
| 10000 | 超越三界 |

## 安装

```bash
cd ~/.astrbot/data/plugins
git clone https://github.com/Nya-Nya-Hoshino/astrbot_plugin_muyu.git muyu
```

## 配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| muyu_enabled | bool | true | 启用赛博木鱼 |
